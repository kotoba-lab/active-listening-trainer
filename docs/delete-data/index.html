端末内データの削除

練習ログ・APIキー・ユーザー辞書などの端末内データは、アプリをアンインストールすることで完全に削除されます。

アプリ内の「履歴を削除」機能からも練習ログを手動で削除できます。

Claude API（Anthropic）に送信したデータ

採点実行時に回答テキストが Anthropic 社のサーバーに送信されます。 このデータの削除については Anthropic 社のプライバシーポリシー に従います。

詳細は https://www.anthropic.com/legal/privacy をご確認ください。

その他のリクエスト

上記以外の削除リクエストやご不明点は、GitHub の Issues からご連絡ください。 できる限り対応いたします。
