プライバシーポリシー

アプリ名：アクティブリスニング トレーナー　／　開発者：melofla8os-ai　／　最終更新：2026年4月1日

1. 基本方針

本アプリは、ユーザーが個人的な傾聴スキルを練習するためのトレーニングツールです。 個人を特定できる情報（氏名・住所・メールアドレス等）は一切収集しません。

2. 取り扱うデータ

音声データ

マイク入力から Android 標準の音声認識 API（Google）を通じてテキストへ変換します。 変換後のテキストはアプリ内の採点処理に使用され、音声そのものは端末内にも外部にも保存しません。

【重要】Claude API への送信について
本アプリは採点機能に Anthropic 社の Claude API を使用しています。 採点を実行すると、ユーザーが入力した回答テキスト（音声認識結果またはテキスト入力）とシナリオ情報が Anthropic のサーバーへ送信されます。

送信は採点ボタンを押したときのみ行われます。送信データに氏名・住所等の個人情報を含めないよう注意してください。
Anthropic のプライバシーポリシーについては https://www.anthropic.com/legal/privacy をご確認ください。

Anthropic API キー

採点機能を使用するにはユーザー自身が取得した Anthropic API キーが必要です。 入力されたキーは 端末内の SharedPreferences にのみ保存され、 開発者を含む第三者が参照することはありません。

練習ログ・学習記録

以下のデータが端末内にのみ保存されます（外部送信なし）。

  • 練習日時・シナリオ名・プレイモード・スコア（SharedPreferences、最大200件）
  • シナリオ別の挑戦回数・合否（Room データベース）
  • ユーザー辞書（独自フレーズ、SharedPreferences）

3. 収集しない情報

  • 氏名・住所・電話番号・メールアドレス等の個人識別情報
  • 位置情報
  • 端末の連絡先・カレンダー・写真
  • クラッシュログや利用統計

4. 外部サービス

  • Android 標準音声認識 API（Google）：音声をテキストに変換するために使用。 Googleのプライバシーポリシーが適用される場合があります。
  • Anthropic Claude API：回答テキストの採点に使用。 採点実行時のみ回答内容がAnthropicに送信されます。

5. データの保持と削除

端末内に保存したデータはアプリをアンインストールすることで完全に削除されます。 アプリ内の「履歴を削除」機能からも練習ログを手動で消去できます。

6. 対象年齢

本アプリは13歳未満を対象としていません。

7. ポリシーの変更

本ポリシーを変更する場合、このページを更新します。 重大な変更がある場合はアプリのリリースノートでもお知らせします。
