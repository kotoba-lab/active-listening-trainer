1. 基本方針（エビデンスベース）

単なる座学ではなく、短時間・反復・フィードバック付き（deliberate practice）が有効

技能は分解して練習する：

  • 感情検出
  • リフレクション
  • 要約

2. 二層構造モデル

Layer 1：アプリ

  • 型の習得
  • パターン認識
  • NG理解

Layer 2：生練習

  • 実際の対人反応
  • 感情負荷の中で実行
  • フィードバックと修正

最適なループ（詳細はSection 7参照）：

アプリ → 生練習 → 現場実践 → 振り返り → アプリ

3. コア練習：感情ピーク＋リフレクション（D練習）

D-0. 書くリフレクション（入門→初級ブリッジ）

D（口頭）の前に「書く」という低負荷形式で初めてリフレクション文を生成する。認知負荷を段階的に上げる設計（Cognitive Load Theory）。

  1. 話し手が2分の実体験を話す
  2. 聞き手は「感情名＋1文リフレクション」を紙に書く（口頭では言わない）
  3. 書いたものを話し手に見せる
  4. 話し手が「合ってた / 近かった / 外れてた」と一言で答える
  5. 外れた場合、話し手が正解に近い感情名を伝える（30秒）
「うまく書けるか」より「何に詰まるか」を観察させること。詰まった点がDの課題になる。

D. コア練習：感情ピーク＋1文リフレクション

ワーク目的

  • 感情検出精度の向上
  • 非助言的関わりの習得
  • セッション終結の基本形（P簡易版）まで一連の流れを体験する

受け手（聞き手）

  • 最後まで遮らずに聞く
  • 感情のピークを1つ決める
  • 感情＋強度を推定する
  • 1文でリフレクションを返す
このワークでの禁止事項
質問・助言・問題解決
※「すぐに何かしたい」という欲求を意図的に抑制することが目的。聞くだけで関わる体験を作るために設定している。

語り手（話し手）

  • 2〜3分の実体験を話す
  • リフレクションを以下の軸で評価する：
    • わかってもらえた感（0〜10）
    • 話したくなったか（Yes / No）
    • ズレた場合は修正を提示する

観察者（任意）

① 構造
  • ルール遵守：OK / NG
② 精度
  • ピーク：前半 / 中盤 / 後半
  • 感情：〇〇寄りに見えた
  • 強度：弱 / 中 / 強
③ 影響
  • 反応：話が広がった / 変化なし / 止まった
  • 気になった点：〇〇

ワークの流れ

  1. 話す（2〜3分）
  2. リフレクション（1回）
  3. 話し手フィードバック（わかってもらえた感・話したくなったか・ズレ修正）
  4. 観察者フィードバック（3軸）
  5. 終結セット（P簡易版）：要約1行＋「話してくれてありがとう」
  6. 役割交代
1セット5〜6分 × 3回＝15〜18分研修

4. Deliberate Practice 設計

基本原則

  • 1技能に絞る
  • 短時間で回す
  • 即フィードバック
  • 反復する

推奨分解（v2）

レベル練習内容
入門感情検出・識別（A→B→C）
初級リフレクション生成（D-0→D）

初級メタ衝動の自覚（I・L）

上級介入・終結（M・N・P）

5. 精度を上げる工夫

  • ピーク1つ縛り：曖昧な共感を防ぐ
  • ズレ歓迎：修正こそ学習の本体（G練習）
  • 深層感情を見る：怒り → 不安・傷つき（E練習）
  • Affirmation を加える：感情反射に加え、話し手の努力・強さを一言認める（Q練習）
    • 例：「それは辛かったよね（リフレクション）。それでも続けてきたんだね（Affirmation）」
    • 褒め言葉ではなく、話し手の語りから引き出した事実として提示すること
  • 自然な言語：教科書表現禁止
    • NG：「あなたは〜と感じているのですね」
    • OK：「それ、しんどかったんだね」
  • 具体化を促す：漠然とした語りに対し「どの瞬間にそう感じた？」など1〜2回まで（R練習）

6. よくある失敗

  • 共感テンプレ化（F練習で対処）
  • 問題解決に飛ぶ（L練習で対処）
  • 感情を広く取りすぎる
  • 自己開示で焦点を自分に移す（I練習で対処）
  • Affirmationが「お世辞」になる ― 語りの中の事実を根拠にすること
  • 具体化質問が尋問化する ― 1回の話で最大1〜2回が上限（R練習）

6-2. v2 新規追加練習の要点

Q. Affirmation（肯定・承認）― 初級

OARS の A（Miller & Rollnick, 2013）に基づく。感情反射（リフレクション）と組み合わせて、話し手の努力・強さを存在として認める1文を返す。

リフレクション → Affirmation の順序を意識させること。
感情を受け取ってから承認する。逆にすると「慰め」に見える。
「ちゃんと見てもらえた感」0〜10で評価。「よく頑張ったね」より「あれだけの状況で動き続けたんだね」（語りから拾う）。

R. 具体化促進（明確化質問）― 中級

話し手が抽象的・曖昧な表現を使っているとき、具体化を促す問いかけを行う（OARS のオープン質問・clarification type）。

  • OK：「それって、例えばどんなことが起きたの？」「どの瞬間にそう感じた？」
  • NG：「なぜそう感じたの？」（原因追及に転化しやすい）
1回の話で最大1〜2回が目安。問いかけ後に必ずリフレクションを1文返すこと。

O. 感情引き込まれチェック（補助ツール・全レベル横断）

練習の前後に自己の感情・身体状態を0〜10で記録する。高感情負荷練習（E・N）では前後に必ず実施。スコアが3以上上昇した場合は5分間休憩を取る。

記録は評価ではなく観察。スコアが高いこと自体は「共感できている証拠」でもある。問題は気づかないことにある（Figley, 1995）。

7. 実務への転移ステップ

Section 2のループの詳細版：

  1. アプリで練習（型の習得）
  2. 生練習で実行（感情負荷あり）
  3. 現場で1回使う（実戦）
  4. 振り返り（何がズレたか）
  5. アプリで再練習（修正点を集中練習）

8. この訓練の本質

  • 正解を当てるゲームではない
  • 相手の反応で調整する技術
  • 「伝わったか」が唯一の評価軸
「感情を1つ捉えて、1文で返し、反応で修正する」
