アクティブリスニング練習　ファシリテーターガイド

全体設計・進行手順・教育的背景　/ 進行者用

1. 基本方針（エビデンスベース）

単なる座学ではなく、短時間・反復・フィードバック付き（deliberate practice）が有効

技能は分解して練習する：

  • 感情検出
  • リフレクション
  • 要約

2. 二層構造モデル

Layer 1：アプリ

  • 型の習得
  • パターン認識
  • NG理解

Layer 2：生練習

  • 実際の対人反応
  • 感情負荷の中で実行
  • フィードバックと修正

最適なループ（詳細はSection 7参照）：

アプリ → 生練習 → 現場実践 → 振り返り → アプリ

3. コア練習：感情ピーク＋リフレクション

ワーク目的

  • 感情検出精度の向上
  • 非助言的関わりの習得

受け手（聞き手）

  • 最後まで遮らずに聞く
  • 感情のピークを1つ決める
  • 感情＋強度を推定する
  • 1文でリフレクションを返す
このワークでの禁止事項
質問・助言・問題解決
※「すぐに何かしたい」という欲求を意図的に抑制することが目的。聞くだけで関わる体験を作るために設定している。

語り手（話し手）

  • 2〜3分の実体験を話す
  • リフレクションを以下の軸で評価する：
    • わかってもらえた感（0〜10）
    • 話したくなったか（Yes / No）
    • ズレた場合は修正を提示する

観察者（任意）

① 構造
  • ルール遵守：OK / NG
② 精度
  • ピーク：前半 / 中盤 / 後半
  • 感情：〇〇寄りに見えた
  • 強度：弱 / 中 / 強
③ 影響
  • 反応：話が広がった / 変化なし / 止まった
  • 気になった点：〇〇

ワークの流れ

  1. 話す（2〜3分）
  2. リフレクション（1回）
  3. フィードバック（1分）
  4. 役割交代
1セット5分 × 3回＝15分研修

4. Deliberate Practice 設計

基本原則

  • 1技能に絞る
  • 短時間で回す
  • 即フィードバック
  • 反復する

推奨分解

レベル練習内容
初級感情ピーク検出のみ
中級感情＋リフレクション
上級深層感情＋修正反射

5. 精度を上げる工夫

  • ピーク1つ縛り：曖昧な共感を防ぐ
  • ズレ歓迎：修正こそ学習の本体
  • 深層感情を見る：怒り → 不安・傷つき
  • 自然な言語：教科書表現禁止
    • NG：「あなたは〜と感じているのですね」
    • OK：「それ、しんどかったんだね」

6. よくある失敗

  • 共感テンプレ化
  • 問題解決に飛ぶ
  • 感情を広く取りすぎる

7. 実務への転移ステップ

Section 2のループの詳細版：

  1. アプリで練習（型の習得）
  2. 生練習で実行（感情負荷あり）
  3. 現場で1回使う（実戦）
  4. 振り返り（何がズレたか）
  5. アプリで再練習（修正点を集中練習）

8. この訓練の本質

  • 正解を当てるゲームではない
  • 相手の反応で調整する技術
  • 「伝わったか」が唯一の評価軸
「感情を1つ捉えて、1文で返し、反応で修正する」
